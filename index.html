DOM stands for document object model and it is the text we get from our html files parsed to a structure of element nodes and text nodes. Element nodes are our html elements and tet nodes represent text and whitespace.

Selecting Elements

Element nodes can be selected via query methods from the document object. These typically either return a single node e.g. querySelector, getElementById or a node list of element nodes e.g. querySelectorAll, getElementsByTagName.

Manipulating Elements

Once you have a dom element selected via a query you have a bunch of properties you can access and change the values of. For example you can change the elements className via the className property.

Traversing the DOM

Traversing the DOM means to go from a selected element via a query selector for example and then from there going to a parent, child, sibling, descendant or ancestor element.

Selecting Child Elements

When you have an element selected you can then use methods and properties to select a specific child element/node or get a list of all the child elements/nodes.

Selecting parent and ancestor elements

When you have an element selected you can use methods and properties to select up the tree to either the elements parent or an ancesotr element higher up the DOM tree.

Selecting sibling elements

When you have an element selected you can use methods and properties to select the previous or next sibling element.

Styling elements

Element styling can be overriden in JavaScript by modiyfing property values on an element such as the style property for inline styling, className for the list of assigned css classes and the classList property object to add, remove and toggle css classes.
